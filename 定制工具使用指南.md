# 网站定制工具使用指南 🎨

欢迎使用臻藏拍卖网站定制工具！这个强大的可视化工具让您可以实时定制网站的各个方面。

## 🚀 快速开始

1. 点击导航栏的 **"网站设置"** 链接
2. 选择一个预设主题或自定义设置
3. 点击 **"应用设置"** 保存更改
4. 所有设置自动保存到浏览器，刷新页面仍然生效

## 🎨 功能详解

### 1. 预设主题（一键切换）

提供 6 种精心设计的主题，点击即可应用：

| 主题名称 | 风格描述 | 适用场景 |
|---------|---------|---------|
| **默认渐变** | 紫蓝渐变，现代科技感 | 通用场景 |
| **海洋蓝** | 深蓝渐变，沉稳专业 | 商务展示 |
| **日落橙** | 暖色调，热情活力 | 活动促销 |
| **森林绿** | 清新自然，环保健康 | 艺术展览 |
| **皇家紫** | 高贵典雅，奢华感 | 高端拍卖 |
| **暗夜黑** | 极简低调，神秘感 | 私密拍卖 |

**使用方法**：点击任意主题卡片，系统自动应用并保存

### 2. 背景设置

#### 渐变色背景
- **渐变起始色**：设置渐变的第一种颜色
- **渐变中间色**：设置渐变的过渡颜色
- **渐变结束色**：设置渐变的最后颜色
- **效果**：从左上到右下的平滑渐变

#### 纯色背景
- 选择单一背景颜色
- 适合简约风格

#### 图片背景
- 输入图片URL（建议使用 Unsplash）
- 图片自动填充并固定
- 推荐分辨率：1920x1080 以上

**示例 URL**：
```
https://images.unsplash.com/photo-1561214115-f2f134cc4912?w=1920&q=80
https://images.unsplash.com/photo-1578926314433-e2789279f4aa?w=1920&q=80
```

### 3. 主题配色

精细控制网站的每一种颜色：

- **主色调**：按钮、链接的主要颜色
- **次色调**：渐变的辅助颜色
- **强调色**：价格、重要信息的颜色
- **成功色**：成功状态、已报名标记
- **危险色**：警告、错误提示
- **警告色**：延时、注意事项

**配色建议**：
- 使用互补色增加视觉冲击
- 保持对比度便于阅读
- 主色调选择品牌色

### 4. 首页文案定制

完全自定义网站文字内容：

```
网站名称      → 导航栏左上角显示
英雄标题      → 首页大标题（例："探索艺术的永恒魅力"）
英雄副标题    → 首页副标题（例："珍稀艺术品，等待与您相遇"）
行动按钮文字  → 首页主按钮（例："立即探索"）
精选区域标题  → 首页拍卖区标题（例："精选拍卖"）
精选区域副标题 → 拍卖区副标题（例："当前最受瞩目的艺术珍品"）
```

**场景示例**：

**个人收藏站**：
```
网站名称: "我的艺术收藏"
英雄标题: "用心收藏每一件艺术品"
```

**企业拍卖**：
```
网站名称: "XX集团拍卖会"
英雄标题: "诚信为本 · 专业拍卖"
```

### 5. 排版设置

#### 卡片圆角 (0-32px)
- **0px**：方形卡片，现代简约
- **8px**：轻微圆角，温和
- **16px**：中等圆角（默认）
- **24px+**：圆润可爱风格

#### 卡片间距 (8-48px)
- **8-16px**：紧凑布局
- **24-32px**：标准布局（默认）
- **40-48px**：宽松留白

#### 标题字体大小 (32-72px)
- **32-40px**：低调内敛
- **48-56px**：标准（默认）
- **64-72px**：霸气醒目

#### 正文字体大小 (14-20px)
- **14-15px**：紧凑精致
- **16px**：标准易读（默认）
- **18-20px**：舒适大字

#### 卡片阴影强度
- **无**：扁平设计
- **轻**：微妙阴影
- **中**：标准阴影（默认）
- **强**：强烈立体感

#### 动画速度
- **快速** (0.2s)：干脆利落
- **正常** (0.3s)：默认
- **缓慢** (0.5s)：优雅舒缓

### 6. 布局设置

#### 卡片列数（桌面端）
- **2列**：大图展示，适合高清图片
- **3列**：默认，平衡美观
- **4列**：紧凑展示，适合多商品

#### 导航栏样式
- **毛玻璃**：半透明模糊效果（默认）
- **实色**：纯色背景，稳重
- **透明**：完全透明，沉浸式

#### 卡片悬停效果
- **上浮**：向上移动 + 缩放（默认）
- **缩放**：只放大
- **发光**：光晕效果
- **无**：无动画

### 7. 高级设置

#### 英雄区域背景图
- 自定义首页大图背景
- 建议尺寸：1920x1080
- 自动添加渐变遮罩

#### 页脚背景色
- 单独控制页脚颜色
- 建议使用深色

#### 自定义CSS
- 直接输入CSS代码
- 高级用户专用
- 可覆盖任何样式

**示例代码**：
```css
/* 添加自定义字体 */
body {
    font-family: "Microsoft YaHei", sans-serif;
}

/* 修改按钮样式 */
.btn-primary {
    border-radius: 20px;
    font-weight: bold;
}

/* 添加特殊效果 */
.auction-card {
    border: 2px solid gold;
}
```

## 💾 配置管理

### 应用设置
点击 **"应用设置"** 按钮：
- 立即应用所有更改
- 自动保存到浏览器
- 刷新页面不丢失

### 重置默认
点击 **"重置默认"** 按钮：
- 清除所有自定义
- 恢复初始设置
- 需要刷新页面

### 导出配置
点击 **"导出配置"** 按钮：
- 下载 JSON 配置文件
- 文件名格式：`auction-site-config-[时间戳].json`
- 可备份或分享

### 导入配置
点击 **"导入配置"** 按钮：
- 选择之前导出的 JSON 文件
- 自动应用配置
- 页面自动刷新

## 🔧 常见问题

**Q: 设置没有生效？**
```
1. 检查是否点击"应用设置"
2. 刷新页面查看效果
3. 清除浏览器缓存后重试
```

**Q: 如何恢复默认设置？**
```
点击"重置默认"按钮即可
```

**Q: 配置可以在其他电脑使用吗？**
```
1. 导出配置文件
2. 在新电脑导入配置
3. 自动应用所有设置
```

**Q: 自定义CSS不生效？**
```
1. 检查CSS语法是否正确
2. 使用 !important 提高优先级
3. 查看浏览器控制台错误信息
```

**Q: 背景图片不显示？**
```
1. 确认URL可访问
2. 使用HTTPS协议的图片
3. 推荐使用 Unsplash 图片
```

**Q: 怎么分享我的配置？**
```
1. 导出配置文件
2. 分享JSON文件给朋友
3. 对方导入即可使用相同设置
```

## 💡 最佳实践

### 配色原则
1. **对比度**：确保文字清晰可读
2. **一致性**：同类元素使用相同颜色
3. **品牌色**：主色调使用品牌颜色
4. **辅助色**：不超过3种主要颜色

### 排版建议
1. **层次分明**：标题与正文大小差异明显
2. **适度留白**：卡片间距不要太密集
3. **圆角统一**：保持风格一致
4. **阴影克制**：不要过度使用

### 性能优化
1. **图片压缩**：使用压缩后的背景图
2. **动画适度**：不要设置过慢的动画
3. **定期清理**：导出配置后重置测试

### 安全提示
1. **定期备份**：导出配置文件保存
2. **测试修改**：重大改动前先导出当前配置
3. **谨慎CSS**：自定义CSS可能影响功能

## 📚 进阶技巧

### 创建多套主题
```
1. 设置主题A → 导出为 config-A.json
2. 设置主题B → 导出为 config-B.json
3. 需要切换时导入对应配置
```

### CSS高级定制

**添加自定义动画**：
```css
@keyframes myAnimation {
    from { opacity: 0; }
    to { opacity: 1; }
}

.auction-card {
    animation: myAnimation 1s ease;
}
```

**修改特定元素**：
```css
/* 修改导航栏 */
.nav-brand {
    font-size: 2rem !important;
}

/* 修改拍卖卡片 */
.auction-card:nth-child(1) {
    border: 3px solid gold;
}
```

**响应式调整**：
```css
@media (max-width: 768px) {
    .hero-title {
        font-size: 32px !important;
    }
}
```
