## 📋 部署步骤

### 第 1 步：创建 GitHub 仓库（2 分钟）

1. 访问：https://github.com
2. 登录您的 GitHub 账号
3. 点击右上角 **+** → **New repository**
4. 填写：
   - 名称：`auction-game`
   - 选择：**Public** ✅
   - **不要**勾选 README
5. 点击 **Create repository**
6. **复制仓库地址**（类似：`https://github.com/你的用户名/auction-game.git`）

---

### 第 2 步：推送代码（1 分钟）

**在终端中执行**（替换 `你的用户名`）：

```bash
git remote add origin https://github.com/你的用户名/auction-game.git
git branch -M main
git push -u origin main
```

**执行 `git push` 时**：
- 浏览器会自动打开
- 登录 GitHub（OAuth，不需要密码）
- 授权后代码自动上传

---

### 第 3 步：启用 GitHub Pages（2 分钟）

1. 打开仓库：`https://github.com/你的用户名/auction-game`
2. 点击 **Settings** → **Pages**
3. Source 选择：**main** 分支，**/ (root)** 目录
4. 点击 **Save**
5. 等待 1-2 分钟
6. 访问：`https://你的用户名.github.io/auction-game/`

## ❓ 常见问题

### Q: 需要密码吗？
**A**: 不需要！使用 OAuth 授权（浏览器登录）

### Q: git push 失败？
**A**: 
- 确保已创建 GitHub 仓库
- 检查仓库地址是否正确
- 如果要求密码，使用 Personal Access Token

### Q: 找不到 Pages？
**A**: 确保仓库是 **Public**（公开）
